## Step-by-Step Process

### Step 1: Product Identification

**If user provides:**

**Product name only** (e.g., "iPhone")
‚Üí Ask: "Which specific model? (e.g., iPhone 15 Pro, iPhone 14, etc.)"

**Model number** (e.g., "SM-G998B")
‚Üí Search directly for this model

**Serial number**
‚Üí First identify the product, then search for deals

**Generic description** (e.g., "a good laptop")
‚Üí Help narrow down: "What's your budget? Any brand preferences? Intended use?"

### Step 2: Initial Discovery (2-3 searches)

Conduct broad searches to understand the market:

```
Search 1: "[product] deals UK"
Search 2: "[product] price comparison Europe" 
Search 3: "[product] discount [current month]"
```

**Analyze results for:**
- Price range (low to high)
- Retailers stocking the product
- Active promotions mentioned
- Price comparison sites found

### Step 3: Deep Dive (3-5 additional searches)

Based on initial findings, target specific sources:

```
Search 4: Visit price comparison site URLs found
Search 5: "[retailer name] [product] offer"
Search 6: "[product] voucher code"
Search 7: Check manufacturer website
```

### Step 4: Deal Verification

For each promising deal:
- ‚úÖ Verify current availability (in stock?)
- üìÖ Note expiration dates (valid until when?)
- üìù Check conditions (minimum purchase, exclusions?)
- üí∞ Validate pricing (VAT included? Shipping costs?)

### Step 5: Organize and Present

Sort findings by value and present clearly

## Troubleshooting

### If No Results Found:
1. Try alternate product names/spellings
2. Search by manufacturer part number
3. Check if product is discontinued
4. Look for successor models
5. Search in different countries

### If Prices Seem Off:
1. Verify VAT is included
2. Check if shipping is extra
3. Look for bundle pricing
4. Confirm currency is correct
5. Check if price is for variant/different spec

### If Links Don't Work:
1. Search product directly on retailer site
2. Provide search term to use
3. Note that availability changed
4. Suggest alternative retailers
